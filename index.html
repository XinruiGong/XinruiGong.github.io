Motion-Refined DINOSAUR for Unsupervised Multi-Object Discovery
Xinrui Gong*, Oliver Hahn*, Christoph Reich, Krishnakant Singh, Simone Schaub-Meyer, Daniel Cremers, Stefan Roth
ICCVW, 2025
project page / arXiv

We present MR-DINOSAUR, a fully Unsupervised Multi-Object Discovery (MOD) method. We generate unsupervised pseudo labels by retrieving video frames without camera motion, for which we perform motion segmentation. We extend the object-centric learning model DINOSAUR using these pseudo labels to unsupervised MOD. MR-DINOSAUR achieves strong multi-object discovery results on the TRI-PD and KITTI datasets, outperforming the previous state-of-the-art despite being fully unsupervised.
